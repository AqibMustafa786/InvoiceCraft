# **App Name**: InvoiceCraft

## Core Features:

- Invoice Form: UI form for creating invoices, including fields for company, client, items, tax, and discounts.
- Live Preview: Real-time invoice rendering that displays a live preview as the user fills out the form.
- Dynamic Logo Handling: Display logo or company name using provided logic; use of <h1> tags kept for visual reasons.
- Invoice Export: Export invoices as PDF files for easy sharing and printing.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust.
- Background color: Light gray (#F5F5F5) for a clean and modern look.
- Accent color: Orange (#FF9800) to highlight key actions and call-to-actions, offering a friendly, approachable feel.
- Body and headline font: 'PT Sans' (sans-serif) to provide a modern yet readable text.
- Clean and structured layout with clear sections for header, client information, items, and totals. Mobile-responsive design adapting to different screen sizes. Spacing and padding meticulously chosen.
- Use a set of professional, consistent icons. Favor flat, geometric designs that complement the 'PT Sans' font choices.
